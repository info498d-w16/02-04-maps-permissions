# 02-04 Maps & Permissions

Code demoing Google Maps, Permissions, and File usage

In order to use the provided Maps API key, you will need to copy the included `debug.keystore` file into your Android home directory.

In the terminal, navigate into the project file. Then

On Mac:
```
cp debug.keystore ~/.android
```

On Windows in Git Bash:
```
cp debug.keystore C:\Users\{user.name}\.android\
```

On Windows in DOS Shell:
```
COPY debug.keystore C:\Users\{user.name}\.android\
```
